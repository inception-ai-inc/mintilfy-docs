### **Setup Instructions**

1. Install the KIlo Code extension in VS Code
2. In the Kilo Code Settings, set the following: API Provider="Inception" and Base URL="https://api.inceptionlabs.ai/v1"
3. Add your API key and set `Model="mercury"`
4. Under Model Configurations, set the Context Window Size to 128,000.
5. Under Advanced Settings, set Tool Call style to JSON (experimental)
