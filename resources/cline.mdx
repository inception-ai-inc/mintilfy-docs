### **Setup Instructions**

1. Install the Cline extension in VS Code
2. In the Cline Settings, set the following: Provider="OpenAI Compatible" and Base URL="https://api.inceptionlabs.ai/v1"
3. Add your API key and set `Model="mercury-coder"`
4. Under Model Configurations, set the Context Window Size to 128,000

[Documentation](https://docs.cline.bot/)
