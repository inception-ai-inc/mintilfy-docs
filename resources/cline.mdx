### **Setup Instructions**

There are two ways to integrate Mercury Coder with Cline.

**Using the OpenRouter provider**
1. Install the Cline extension in VS Code
2. Click the gear icon (⚙️) in the top-right corner
3. Set the API Provider = “OpenRouter”
4. Enter your OpenRouter API Key
5. Set the Model = “inception/mercury-coder”
6. Click the Done button on the top-right.
