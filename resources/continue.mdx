### **Setup Instructions**

1. Install the Continue extension for VS Code or JetBrains.
2. Sign into your Continue account
3. Search for Mercury Coder and select the Mercury Coder Block


4. Click "Add block" to add the block
5. Click the Block inputs button and enter your API key as `INCEPTION_API_KEY`
6. Save and then Publish Changes
7. In VS Code or JetBrains, set Mercury Coder as the active model for your current Continue session.
