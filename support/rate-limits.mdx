---
title: "Rate Limits"
---

API requests can be subject to rate limiting based on usage and GPU availability. Your requests can sometimes take longer than expected due to the high demand.

Please [contact support](https://platform.inceptionlabs.ai/support) if you need increased rate limits.

|                   | Limit Per Minute |
| :---------------: | ---------------: |
| **API Requests**  | 400              |
| **Input Tokens**  | 400,000          |
| **Output Tokens** | 100,000          |
